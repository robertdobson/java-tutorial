What is a loop?

A loop is a sequence of statements which is specified once but which may be carried out several times in succession. The code "inside" the loop (the body of the loop) is obeyed a specified number of times, or once for each of a collection of items, or until some condition is met, or indefinitely.

A loop is demonstrated with while statements in the Code Demo GUI using the following code from the loopDemoMethod:


public static void loopDemoMethod(int userEntry) {

        while(userEntry > 49) {
            userEntry -= 50;
            ++halfDollars;
        }

        while(userEntry > 24) {
            userEntry -= 25;
            ++quarters;
        }

        while(userEntry > 9) {
            userEntry -= 10;
            ++dimes;
        }

        while(userEntry > 4) {
            userEntry -= 5;
            ++nickels;
        }
        CodeDemoGUI.showResult.setText("Your change is " +
                halfDollars + " half dollar(s), " + quarters +
                " quarter(s), " + dimes + " dime(s), " + nickels +
                " nickel(s), and " + userEntry + " penn(y/ies)!");

        // Reset global variables
        halfDollars = 0;
        quarters = 0;
        dimes = 0;
        nickels = 0;
    }
